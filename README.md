This is the backend for YourJob.com, built with nodejs, typescript and using express as framework.

### Libraries in use

- Bcryptjs
- Cors
- Dotenv
- Ejs
- Google-libphonenumber
- Html-to-text
- Jsonwebtoken
- Juice
- Nodemailer
- Sequelize
